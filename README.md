# Caicedo_Pearce Project

* BEFORE EDITING
    1. PULL THE LAST VERSION OF THE REPOSITORY
a) Before starting to edit any file, open GitHub Desktop and click on “fetch original” or “pull master”

* AFTER EDITING
    1. MAKE “COMMIT TO MAIN”
a) Open GitHub Desktop
b) On the left (in the changes tab) you will see the list of files you edited and the changes you made
c) Check those files are the ones you edited
d) In the bar below,  write a heading that describes the changes you made. 
e) In the Description box, describe precisely what the change consists of.

2. PUSH THE CHANGES TO THE REPOSITORY
a) At the top click on push origin
